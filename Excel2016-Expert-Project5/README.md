# Description
你在答得喵云数据服务公司工作。你正在创建一个数字表格帮助你分析数据并制定市场计划。

# Task 1
在“订阅者”工作表的F列，使用一个公式，使得如果用户超过了他们的用量限制显示TURE。否则，显示FALSE。你的公式中必须使用AND和OR函数。

# Task 1 Answer
<details>
  <summary>Click to see answer</summary>
`=IF(AND(D2="白金",E2>$I$3),AND(D2="黄金",E2>$I$4),"是","否")`
